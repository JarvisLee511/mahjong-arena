# 牌面來源 / Tile art source

平面(Planar)麻將牌面 SVG,取自 Wikimedia Commons:
**Category: SVG Planar illustrations of Mahjong tiles**
https://commons.wikimedia.org/wiki/Category:SVG_Planar_illustrations_of_Mahjong_tiles

- 檔名對應(原檔名 → 本專案代碼):
  - `0101一萬`..`0109九萬` → `m1`..`m9`
  - `0201一餅`..`0209九餅` → `p1`..`p9`(餅=筒）
  - `0301一條`..`0309九條` → `s1`..`s9`
  - `0401東風`→`z1`、`0403南風`→`z2`、`0402西風`→`z3`、`0404北風`→`z4`
  - `0405中`→`z5`、`0406發`→`z6`、`0407白`→`z7`
  - `0501春`→`f1`..`0508竹`→`f8`
- 下載腳本:`assets/download-flat.js`
- 這些是正面平面圖(非立體),透明背景;牌底(象牙圓角+薄邊)由 CSS `.tile .tface-sym` 繪製。
- 使用/散佈前請至各檔案頁面確認授權條款並依規標註出處(部署公開時)。
