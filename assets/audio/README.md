# 音效音檔清單 🎙️（台語 / 報牌）

把音檔放進**這個資料夾** `assets/audio/`,再把檔名列進 `manifest.json`。
目前內建低音量國語報牌；用同檔名放入你自己的錄音即可替換。
格式支援 **mp3 / m4a / wav / ogg**(手機錄的 m4a 直接丟進來就行)。

## A. 喊話(重點,先錄這 6 個)

| 檔名 | 什麼時候播 | 台語參考 |
|---|---|---|
| `pung`  | 有人碰 | 碰!(pōng) |
| `kong`  | 有人槓 | 槓!(kòng) |
| `chow`  | 有人吃 | 食!(tsia̍h) |
| `hu`    | 放槍胡別人 | 胡矣!(hô--ah) |
| `tsumo` | 自摸 | 自摸!(tsū-bong) |
| `draw`  | 流局 | 流局!/無人胡 |
| `start` | (選)開一局 | 來喔! |
| `tenpai`| (選)聽牌 | 聽牌矣! |

## B. 報牌(每打一張牌念牌名,想做才錄,共 34 張)

檔名就是牌的代碼:

| 代碼 | 牌 | 代碼 | 牌 | 代碼 | 牌 |
|---|---|---|---|---|---|
| `m1`~`m9` | 一萬~九萬 | `p1`~`p9` | 一筒~九筒 | `s1`~`s9` | 一條~九條 |
| `z1` | 東 | `z2` | 南 | `z3` | 西 |
| `z4` | 北 | `z5` | 中 | `z6` | 發 |
| `z7` | 白 | | | | |

## 最後:把檔名填進 manifest.json

```json
["pung.m4a", "kong.m4a", "chow.m4a", "hu.m4a", "tsumo.m4a", "draw.m4a",
 "m1.m4a", "m2.m4a", "z1.m4a", "z5.m4a"]
```

(懶得改 JSON 就把檔案丟進資料夾、跟我說一聲,我幫你加。)
每句剪短一點(1 秒內)、背景安靜最好聽。放進去重整就生效 👍
